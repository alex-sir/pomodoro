Keyboard controls:

  • - SPACE - Play/Pause
  • - ALT + S - Stop (current time reset)
  • - ALT + R - Reset to original values (25/5)
  • - ALT + P - Session
  • - ALT + B - Break
  • - ALT + L - Long Break
  • - ALT + ↑ - Increase time
  • - ALT + ↓ - Decrease time
